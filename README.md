# Reader - manga platform.

## Features

- Registration and authorization.
- Manga(Create,Read).
- Chapters(Create,Read).
- Comments(CRUD)
- Bookmarks
- Teams
- Search

## Technology stack:

- Next js
- Typescript
- Redux Toolkit
- Ant Design
- React-hook-form

## Linting:

- ESLint
- Prettier

## Tests:

- Jest
- React Testing Library
- Cypress

## Install

- 1 Install all packages:

```bash
npm i
# or
yarn install
```

- 2 Start app:

```bash
npm run dev
# or
yarn dev
```

- 3 You need to raise the backend. :
  The backend version [here](https://github.com/pavel-developer2001/reader-backend-nest).
